## MediaGet.js 1.0
本專案為影音平台之影片、音源之取得類別庫實作，可以幫助使用者快速取得指定媒體。

#參考
本專案實作方式參考自[MediaGetCore](https://github.com/XuPeiYao/MediaGetCore)專案。
